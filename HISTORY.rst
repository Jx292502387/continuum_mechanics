History
=======

0.1.0 (2018-04-21)
------------------

* First release on PyPI.

0.1.2 (2019-04-04)
------------------

* Add elasticity operators.

0.2.0 (2019-12-02)

* Extend 2D Mohr circle to asymmetric tensors.

* Add Voigt notation.

* Add Kelvin-Mandel notation.

* Add Christofel stiffness for 4th order tensors.

* Add visualization for 4th order tensors.
